# 서울여자대학교 데이터사이언스학과 사제 동행 소학회 2024 AM.SWU의 Github
선후배가 함께 성장하는 소학회, **`AM.SWU`** 에 오신 것을 환영합니다.

AM.SWU는 서울여자대학교 데이터사이언스학과 소속 사제동행 소학회로, 데이터사이언스학과 소속 학회원들이 함께 노력하며 개인의 지속적인 성장과 발전을 돕고, 함께 ML 관련 프로젝트를 하며 친목 도모를 합니다. 개발 실무를 체득할 수 있도록 함께 스터디를 진행하고 프로젝트를 진행하는 것으로 커리큘럼을 구성하여 2년째 운영되고 있습니다.

이곳에서는 단순한 '역량 개발'이 아니라, 빠른 IT의 변화 속에서 다함께 지속적으로 성장할 수 있는 '좋은 개발자의 경험과 습관'을 가진 지속 가능한 개발자 양성을 목표로 소학회가 운영됩니다. 개인이 아니라 팀 협업을 통해 배우고 데이터 문제를 해결하며 다함께 성장하는 학습 커뮤니티를 지향합니다.

2024 AM.SWU 소학회에 참여한 학생(소학회원)들의 프로젝트 결과가 궁금하다면 여기에서 확인할 수 있습니다.

### [AM.SWU 소개]
```python
newValue = artificialIntelligence + machineLearning
AMSWU = newValue
print('Welcome to', AMSWU, '!')
```

소학회 AM.SWU에서는 인공지능과 머신러닝이 무엇인지를 탐구하고, 탐구한 내용을 바탕으로 더 나은 세상을 만들기 위한 새로운 가치를 발굴하기 위해 다양한 활동을 진행합니다.
현재 소학회원으로 21학번 10명, 22학번 9명, 23학번 4명으로 구성되어 있습니다. 

* 기간 : 2024년 3월 2일 ~ 2024년 12월 21일, 5개월 (학기 중으로 진행 예정)
* 구성 : 스터디 + 팀별 프로젝트
